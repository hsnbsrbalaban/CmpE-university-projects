Also, methods are to be called for current player. For example, when move() is called, you move a piece of the player who has the turn.

  • Method: public boolean move(String from, String to)
     Action: Moves the piece at from to a new position specified in to.
    Input: Both from and to are strings of length 2. The first character is one of 'A,B,C,D,E,F,G,H' and the second char is one of '1,2,3,4,5,6,7,8'.
    Return value: If the move take place, the return value is true; otherwise, false. Observe that, if there is no piece at from, you return false.
    Example:
    ChessFrame theFrame = new ChessFrame();
    theFrame.setVisible(true);
    theFrame.move("b1", "a3");

  • Method: public String at(String pos)
     Action: Returns the information of the piece at pos.
    Input: A string of length 2, where the first character is one of 'A,B,C,D,E,F,G,H' and the second char is one of '1,2,3,4,5,6,7,8'.
    Return value: A string of the form 'xxx-yyy', where 'xxx' denotes the color of the piece (e.g. white/black) and 'yyy' represents the type of the piece (e.g.king/queen/rook/bishop/knight/pawn). If there is no piece at the specified position, return empty string: ''
    Example:
    ChessFrame theFrame = new ChessFrame();
    theFrame.setVisible(true);
    theFrame.at("b1"); // returns "white-knight"

  • Method: public boolean castling(boolean isKingSide)
     Action: Implement kingside and queenside castling. Originally, castling may only be done if the king has never moved, the rook involved has never moved, the squares between the king and the rook involved are unoccupied, the king is not in check, and the king does not cross over or end on a square in which it would be in check. Ignore all of these rules for this project. You may safely assume that castling may be done as long as there the rook and the king are in their original positions, and there is no piece between them. That is to say, castling can occur if the pieces were moved some time during the game and got back their original positions.
    Input: A boolean that is true if the goal is a kingside castling.
    Return value: true, if move is successful; false, otherwise.
    Example:
    ChessFrame theFrame = new ChessFrame();
    theFrame.setVisible(true);
    theFrame.casling(true);

  • Method: public void undo()
     Action: Undos the last move. Note that this method may be used multiple times in a row. So, consider using stack.
    Input: -
    Return value: -
    Example:

    theFrame.setVisible(true);
    // here some moves are done
    theFrame.undo();

  • Method: public boolean isInCheck()
     Action: Checks if the current player's king is threaten with capture.
    Input: -
    Return value: true, if the king is threaten; false, otherwise
    Example:
    ChessFrame theFrame = new ChessFrame();
    theFrame.setVisible(true);
    theFrame.isInCheck();

  • Method: public boolean isCheckmate()
     Action: Checks if the current player has no move to save its king from being captured.
    Input: -
    Return value: true, if the game is over; false, otherwise
    Example:
    ChessFrame theFrame = new ChessFrame();
    theFrame.setVisible(true);
    theFrame.isCheckmate();

  • Method: public void save(String fileName)
     Action: Saves the current status of the game to a file whose path is given as parameter. The first line of the file is the color of the player that has the turn. Each of the following line, having the format 'xxx-yyy-zzz', stores the information of a piece on the table where 'xxx' denotes the color of the piece (e.g. white/black) and 'yyy' represents the type of the piece (e.g.king/queen/rook/bishop/knight/pawn) and 'zzz' is the position (a string of length 2, where the first character is one of 'A,B,C,D,E,F,G,H' and the second char is one of '1,2,3,4,5,6,7,8')
    Input: The path of the file.
    Return value: -
    Example:
    ChessFrame theFrame = new ChessFrame();
    theFrame.setVisible(true);theFrame.save("/home/game.txt");
     The output file looks like this:
    black
    white-rook-a1
    white-knight-b1
    black-pawn-d5
    ...

  • Method: public static ChessFrame load(String fileName)
     Action: Loads the ChessFrame instance previously saved to the file given in parameter
    Input: The path of the file to be loaded.
    Return value: The ChessFrame instance, i.e., the game itself.
    Example:
    ChessFrame theFrame = ChessFrame.load("/home/game.txt");
